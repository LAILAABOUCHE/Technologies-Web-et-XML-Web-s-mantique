  • Data Visualization avec Python L’idée est de développer une application qui analyse et transforme des données brutes en graphiques compréhensibles. En utilisant des bibliothèques comme Matplotlib, Seaborn ou Plotly, le projet permet de créer des visualisations (histogrammes, courbes, diagrammes) interactives. Cela aide à détecter des tendances et des corrélations de manière visuelle et intuitive.

  • Projet RAG en Java Ce projet consiste à créer une application en Java qui automatise la génération de rapports à partir de données collectées en temps réel. L’application récupère des données depuis différentes sources, les traite, puis génère et affiche des rapports clairs via une interface graphique (par exemple avec Swing). L’objectif est de faciliter le suivi et l’analyse des information.

  • Reconnaissance Faciale Ce projet vise à développer une application capable de détecter et identifier des visages dans des images ou vidéos. À l’aide d’outils comme OpenCV et éventuellement des modèles d’apprentissage automatique, l’application peut comparer des caractéristiques faciales avec une base de données pour authentifier des individus. Ce système peut être utilisé pour la sécurité ou le contrôle d’accès.
